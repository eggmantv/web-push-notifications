## 课程说明

为了方便在每节课的代码间进行切换，该课程是通过git tag的方式来组织的，就是每节课对应一个git tag。

首先clone该项目

```shell
$git clone git@github.com:eggmantv/tdd_in_action.git
```

切换tag
```shell
$cd tdd_in_action

查看所有标签
$git tag
01
02
...

切换到第一节课的源码
$git checkout 01
